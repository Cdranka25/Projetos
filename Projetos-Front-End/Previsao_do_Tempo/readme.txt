# Previsão do Tempo

Este é um projeto simples de previsão do tempo usando a API do OpenWeather. Ele permite que o usuário pesquise a previsão do tempo para uma cidade específica ou use sua localização atual para ver as condições meteorológicas. O projeto também exibe uma previsão para os próximos 5 dias.

## Funcionalidades

- **Pesquisa de cidade**: Permite ao usuário pesquisar o clima de uma cidade digitando o nome.
- **Localização atual**: Permite ao usuário obter a previsão do tempo baseada em sua localização geográfica atual.
- **Previsão do tempo**: Exibe a temperatura, a umidade, a velocidade do vento e uma descrição do clima.
- **Previsão dos próximos 5 dias**: Exibe uma previsão detalhada do clima para os próximos 5 dias.

## Tecnologias Utilizadas

- **HTML5**: Para estruturação da página.
- **CSS3**: Para estilização e layout responsivo.
- **JavaScript**: Para manipulação do DOM e interação com a API do OpenWeather.
- **API do OpenWeather**: Para obter os dados de clima e previsão.
